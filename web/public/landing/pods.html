Deep Pod-Level Troubleshooting

Everything you need to understand what is happening inside your pods, across every cluster, in one place.

🟢

Status Overview

View Running, Pending, Succeeded, Failed, and Unknown pod states across all clusters. Filter by phase, namespace, or node to zero in on problem areas and track patterns over time.

📜

Live Log Streaming

Stream container logs in real time from any pod in any cluster. Support for multi-container pods, init containers, and previous container logs for debugging restart loops.

🔄

Restart Analysis

Track pod restart counts and reasons. Identify CrashLoopBackOff patterns, OOMKilled containers, and liveness probe failures with timelines showing when and why restarts occurred.

📊

Resource Consumption

Compare actual CPU and memory usage against requests and limits for each container. Identify containers hitting their memory ceiling or barely using their allocated CPU.

🔍

Event Correlation

View Kubernetes events associated with each pod: scheduling decisions, image pulls, volume mounts, and probe outcomes. Understand the full lifecycle of a pod from creation to termination.

🌐

Cross-Cluster Pod Search

Find any pod across your entire fleet by name, label, image, or IP address. Instantly locate pods regardless of which cluster or namespace they are running in.
